How It Works

Get started in minutes

1

Launch the Server

Open SentencePlayer MCP and click "Start Server" to begin listening on port 9090

2

Configure Your Client

Add the SSE endpoint to your MCP client: http://127.0.0.1:9090/sse

3

Start Speaking!

Your AI assistant can now use text-to-speech tools to speak responses aloud
