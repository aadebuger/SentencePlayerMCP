Privacy Policy

Your privacy is important to us

Introduction

SentencePlayer MCP Server ("we", "our", or "the App") is committed to protecting your privacy. This Privacy Policy explains how we handle your information when you use our application.

Information We Collect

We do not collect any personal information. SentencePlayer MCP Server operates entirely on your local device and does not transmit any data to external servers.

Data Processed Locally

The following information is processed locally on your Mac and never leaves your device:

  • Text Input: Text you send to the text-to-speech engine
  • Voice Settings: Your selected voices and preferences
  • Server Logs: Connection logs stored locally for debugging purposes
  • Session Information: Temporary session IDs for managing multiple client connections

How We Use Information

All data processing happens entirely on your local machine for the following purposes:

  • Providing text-to-speech functionality using macOS built-in speech synthesis
  • Managing connections between MCP clients and the server
  • Displaying server status and connection information
  • Debugging and troubleshooting through local logs

Data Storage

All data is stored locally on your Mac. The App stores configuration settings and preferences using standard macOS mechanisms:

  • User preferences (voice settings, server port, etc.)
  • Optional debug logs (can be disabled)
  • Session information (temporary, cleared on restart)

You can delete all app data at any time by removing the application from your Mac.

Network Activity

SentencePlayer MCP Server creates a local HTTP/SSE server that listens on localhost (127.0.0.1) by default. This means:

  • The server is only accessible from your Mac
  • No external network connections are made
  • No data is transmitted over the internet
  • Other devices on your network cannot access the server (unless you explicitly configure it)

Third-Party Services

We do not use any third-party services, analytics, or tracking tools. The App does not integrate with any external services except:

  • macOS Speech Synthesis: Built-in system service (handled by macOS, subject to Apple's privacy policy)

Data Sharing

We do not share any data with third parties. Since all processing happens locally and no data is collected, there is nothing to share.

Children's Privacy

Our App does not collect any personal information from anyone, including children under 13. The App is safe for users of all ages.

Your Rights

Since we don't collect your personal information, you have complete control over all data:

  • Access: All data is stored locally and accessible through macOS Finder
  • Deletion: Simply delete the app to remove all associated data
  • Control: You can disable logging and other features at any time

Security

The App follows macOS security best practices:

  • Sandboxed operation (if distributed through Mac App Store)
  • Local-only network access by default
  • No external data transmission
  • Standard macOS file permissions for local storage

Open Source

SentencePlayer MCP Server is open source. You can review our code at GitHub to verify our privacy practices.

Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date.
